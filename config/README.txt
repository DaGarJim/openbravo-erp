======
README
======

Before you can compile, you will need to configure the files XmlPool.xml, dbCon5.xml, log4j.lcf, userconfig.xml with the correct template depending on whether you use ORACLE or PostgreSQL.

In Oracle, setup process replaces parameters and save it with the correct name.

In PostgreSQL instalation you must replace the database hostname, sid and
Tomcat_folder, tomcat_logs and save it with the correct name (as XmlPool.xml, dbCon5.xml, log4j.lcf or userconfig.xml)
